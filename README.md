# code-generate

用于公司项目代码生成，避免反复 copy。

#### 使用

```shell
jee create <name> <path> [options]
```

**Arguments**

| Argument | Description              |
| -------- | ------------------------ |
| name     | 模块名                   |
| path     | 资源输出路径（相对路径） |

**Options**

| Option  | Description        |
| ------- | ------------------ |
| -- type | 模板类型，别名：-t |
